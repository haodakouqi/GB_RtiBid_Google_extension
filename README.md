# RtiBid Pixel Helper

## 项目介绍

RtiBid Pixel Helper 是一款 Chrome 浏览器扩展程序，专门用于监控和验证 RtiBid Pixel 的运行状态。它可以帮助您：

- 检查像素安装是否正确
- 识别并提示错误信息
- 提供实施建议
- 简化像素问题排查流程

## 功能特性

- **实时监控**：在您浏览网站时实时检查像素状态
- **详细报告**：提供像素触发的详细信息和状态
- **错误检测**：自动识别常见的像素安装错误
- **实施建议**：基于检测结果提供优化建议
- **用户友好**：直观的界面设计，易于使用

## 安装指南

### 方法一：从 Chrome 网上应用店安装
1. 打开 Chrome 浏览器
2. 访问 Chrome 网上应用店
3. 搜索 "RtiBid Pixel Helper"
4. 点击 "添加至 Chrome" 按钮
5. 确认安装

### 方法二：从源码安装（开发模式）
1. 克隆或下载本仓库到本地
2. 打开 Chrome 浏览器，进入扩展程序管理页面（chrome://extensions/）
3. 开启右上角的 "开发者模式"
4. 点击 "加载已解压的扩展程序"
5. 选择本项目的根目录
6. 确认加载

## 使用方法

1. 安装扩展程序后，浏览器工具栏会出现 RtiBid Pixel Helper 图标
2. 访问您要检查的网站
3. 点击工具栏中的 RtiBid Pixel Helper 图标
4. 在弹出的面板中查看像素状态和详细信息
5. 根据提示解决发现的问题

## 常见问题

### 扩展程序无法检测到像素
- 确保您的网站已正确安装 RtiBid Pixel
- 刷新页面后再次尝试
- 检查是否有其他扩展程序干扰

### 像素状态显示异常
- 检查像素代码是否完整
- 确认像素 ID 是否正确
- 查看浏览器控制台是否有相关错误信息
